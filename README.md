# 🛍 Product Preview Card Component

A responsive product preview card built with HTML and CSS.

This project is based on a **Frontend Mentor challenge** and focuses on building a clean product card layout with responsive design.

🔗 Live Demo  
https://devdiegorosa.github.io/product-preview-card-component/

---

## 👨‍💻 About the Project

This project displays a product card with image, description, price and a call-to-action button.

The goal of this challenge was to recreate the design as close as possible and practice building responsive UI components using modern CSS techniques.

---

## 🚀 Features

- Responsive layout
- Product image preview
- Product description
- Price display with discount
- Add to cart button
- Clean and modern UI

---

## 🛠 Technologies Used

- HTML5
- CSS3
- Flexbox
- Responsive Design

---

## 🎯 What I Learned

While building this project I practiced:

- Creating responsive components
- Structuring semantic HTML
- Building UI cards
- Using Flexbox for layout
